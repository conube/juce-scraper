Run with AZK
------------

1.	Install AZK

http://docs.azk.io/en/installation

1.	Run the application

`azk start`

Run without AZK
---------------

1.	Install Node.js

	https://nodejs.org/download

2.	Install dependencies

	`npm install`

3.	Run the application

	`node app.js`
